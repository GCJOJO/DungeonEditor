How does Dungeon Editor works?

Keybinds

Edit Mode

-> : Move Right

<- : Move Left

/\ : Move Up

v : Move Down

Space : Place or Destroy

R : Camera Up

T : Camera Down

TAB : Change between Edit Mode and Free Camera

In Game

Z: Forward

S : Backward

Q: Move Left

D : Move Right

F : Interact

(TAB : IF IN EDIT MODE : change mode)
